# Slack-bot
A bot on Slack called 'hughmanns-bot' that can use slash commands.

**Try it out here: [https://app.slack.com/client/E09V59WQY1E/C09PXLPEL2Y](url)**
- Open the demo link.
- Join the workspace.
- Type in a slash command to get started!

## Features / Commands:
- /hughmanns-bot-help: Pulls up a menu of available commands.
- /hughmanns-bot-ping: Tells you your current latency.
- /hughmanns-bot-catfact: Tells you a random cat fact.
- /hughmanns-bot-ping: Tells you a joke.
- The bot is also live 24/7 and can always respond to messages.

## How it works:
- You type in a command (e.g. /hughmanns-bot-help).
- The command is sent to the JavaScript code, which then gives you a response.
- The bot uses a Nest container to keep it running 24/7!
